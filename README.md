## Memo

### install browserMqtt.js

`結局、使ってません`

```
### make package.json
$ mkdir work
$ cd work
$ npm init -y
### install mqtt
$ npm install mqtt -save
### install browserify
$ (sudo) npm install -g browserify
### make browserMqtt.js
$ cd node_modules/mqtt
$ npm install .
$ browserify mqtt.js -s mqtt > browserMqtt.js
$ cp browserMqtt.js ${jsdir}
```
